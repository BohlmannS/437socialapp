Inputting Schedule Instructions:

  1. Login to your account on Webstac and access your schedule
  2. Make sure schedule is in LIST VIEW.
  3. Select All or CTRL+A on Windows or CMD+A on Mac, then copy.
  4. Paste into the input box.
